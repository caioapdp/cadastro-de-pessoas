# 📋 Cadastro de Pessoa — Java

Projeto desenvolvido durante os estudos de Java com foco nos **tipos primitivos**.

## Conceitos utilizados
- Variáveis e tipos primitivos: `int`, `double`, `char`, `boolean`, `String`
- Saída de dados com `System.out.println()`
- Concatenação de Strings

## Como executar
1. Clone o repositório
2. Abra no IntelliJ ou VS Code
3. Execute a classe `CadastroPessoa`
